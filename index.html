c 出现connected即为连接成功。
如果板子上显示C字符即为连接成功。

手把手教你搭建自己的hexo+GitHub博客

发表于 2020-05-20 更新于 2020-05-27 Valine：

为记录自己的搭建过程及所用资料，仅供参考。
视频资源来自哔哩哔哩up主，侵删。

  • 第一步node安装
    a 下载nodejs，并进行安装，建议安装到D盘。
    b 同时按下Windows+r，输入cmd打开命令函，输入node -v按回车（每输入一条密令按回车启动）。
  • 第二步GitHub注册
    a google搜索GitHub，进入官网进行注册账户，并验证邮箱。
    b 点击start a project，repositoryname输入你的账户名.github.io，创建储存库。
  • 第三步git for win安装
    a 下载git for win，建议安装至D盘。
    b 一直点击next至出现三个选项，选择use git from the Windows command prompt接着继续next至安装完成。
    c 打开cmd输入git。
  • 第四步githubSHH配置
    a 输入密令ssh-keygen -t rsa -C “你的邮箱” 。
    b 打开你的github，点击右上方头像旁的箭头选择settings。
    c 点击SSH and GPG keys,新建秘钥，title随便写，打开ssh文件夹，用记事本方式打开id_rsa.pub，复制黏贴进key。
  • 第五步hexo本地使用
    a 百度搜索hexo进入hexo官网，点击文档就可以看见你的安装密令,复制。

    b 打开D盘（个人建议）新建一个blog文件夹，打开cmd输入D:转至D盘，输入cd blog，输入复制的安装密令，等待。
    c 输入hexo init (你的博客名),等待。
    d 进入安装目录（输入cd (你的博客名)），输入密令npm install，等待。
    e 本地预览一下输入密令hexo s -p,复制网址进行预览。
  • 第六步sublime安装
    a 搜索下载sublime安装包，安装至D盘。
    b 把你的博客目录直接拖进打开的subline软件。
    c 打开文件夹里的abc（这里指你的博客名）>source>-post>hello-world,在第6行输入##hello hexo，保存（支持快捷键Ctrl+s）。
  • 第七步hexo发布至github
    a 打开你的github存储库复制此链接。

    b 用sublime打开文件abc>themes>config.yml，在最后面type:输入git。
    c 另起一行输入repo: ，将复制的链接黏贴。
    d 在第10行左右的author：输入你的用户名，在16行左右的url：复制你的博客链接。在博客文件夹下打开右击打开git bash 或打开cmd，cd到你的博客文件夹，输入hexo g回车hexo d回车。
    e 接下来的步骤打字较为麻烦详情点击视频链接观看。之前步骤描述不清晰的话请点击视频反复观看。
